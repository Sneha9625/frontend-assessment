# Frontend Assessment

A responsive frontend assessment project built using HTML, CSS, JavaScript, and Vite.

---

## 🚀 Live Demo

Live Site:https://frontend-assessment-psi-lemon.vercel.app/


GitHub Repository:https://github.com/Sneha9625/frontend-assessment


---

## 📌 Features

- Responsive Hero Slider
- Product Grid Layout
- Tabs Functionality
- Autoplay Slider
- Interactive Navigation Dots
- Keyboard Accessibility
- Responsive Design for Mobile, Tablet, and Desktop
- Hover Effects and Smooth Transitions

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Vite

---

## 📱 Responsive Breakpoints

| Device | Layout |
|---|---|
| Desktop | 4 Product Cards |
| Tablet | 2 Product Cards |
| Mobile | 2 Product Cards |
